What is NPTG?

National Public Transport Gazetteer (NPTG) is an official UK dataset for places the UK Department for Transport and Traveline have offered to make available to the OpenStreetMap project. The NPTG dataset contains information on about 50.000 localities in the UK. Additionally to names and locations for places the gazetteer also provides information for points of interest. However, this information is only patchy and likely to be removed in the future. In case you encounter a POI from NPTG it is best to hide it.

Further information about the NaPTAN/NPTG import can be found in the OpenStreetMap-Wiki.

What is the NPTG-Viewer and what can I do with it?

The NPTG-Viewer allows to compare place nodes in the OSM database with localities in the NPTG dataset. NPTG data has not been imported into the OSM database as its hierarchy does not match the place hierarchy of OSM.

E.g. the viewer can be used to find missing places in OSM, to spot erroneous place names or to find double place names.

When do localities match?

Two localities are considered to be matching if they originate from different datasets (one from OSM and the other one from NPTG), if they have exactly the same names and if they are not too far apart.

When are localities erroneous?

The NPTG-Viewer complains about two types of errors. First, it expects to find no more than one matching locality for each place. Secondly, places taken from OSM are only considered valid if they have a name tag and a place tag set to city, town, municipality, village, hamlet, suburb, island, locality or farm.

What does “hiding”mean?

Localities can be hidden in NPTG-Viewer to mark them as being invalid. Thus, duplicate/double localities or places with invalid tags can be marked. It is planned to provide lists of hidden localities that can be downloaded. These lists can be updated by (?) the OSM database.

When a location is being hidden an additional comment can be made explaining the reason for hiding the locality.

Is the NPTG-Viewer an open-source application?

The NPTG-Viewer is meant to be open-source software. However, I did not manage to publish the source code. However, if you are interested in the source code please e-mail me and I'll be happy to send you the code.
